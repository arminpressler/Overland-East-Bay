Baofeng Programming Cheat Sheet

Master your radio in the field

Note: This guide is designed for the Baofeng UV-5R and similar models. Ensure you are legally licensed to transmit on the frequencies you program.

Quick Reference: Simplex Programming

For vehicle-to-vehicle communication (like our standard 146.460 MHz), follow these steps to save a channel.

  1. Switch to VFO Mode: Press the VFO/MR button until you see a frequency (not a channel number) on the display.
  2. Select Upper Display: Press A/B to move the arrow to the top line.
  3. Delete Existing Channel:
    • Press MENU -> 2 8 -> MENU.
    • Select the channel number (e.g., 001) you want to use.
    • Press MENU to confirm deletion (voice may say "Cancelled" or "Confirmed"), then EXIT.
  4. Enter Frequency: Type the desired frequency on the keypad (e.g., 1 4 6 4 6 0).
  5. Save Channel:
    • Press MENU -> 2 7 -> MENU.
    • Select the same channel number you just deleted (e.g., 001).
    • Press MENU to confirm (voice says "Receiving Memory"), then EXIT.

Quick Reference: Repeater Programming

Repeaters require a transmit offset and often a sub-audible tone (CTCSS/DCS) to open the repeater.

  1. Set Frequency: Follow steps 1-4 above to enter the Receive frequency.
  2. Set Offset Direction:
    • Press MENU -> 2 5 -> MENU.
    • Select + or - based on the repeater info.
    • Press MENU -> EXIT.
  3. Set Offset Amount:
    • Press MENU -> 2 6 -> MENU.
    • Enter the offset (e.g., 00.600 for 2m, 05.000 for 70cm).
    • Press MENU -> EXIT.
  4. Set Transmit Tone (PL):
    • Press MENU -> 1 3 (T-CTCS) -> MENU.
    • Select the tone frequency (e.g., 100.0).
    • Press MENU -> EXIT.
  5. Save Receiving Frequency:
    • Press MENU -> 2 7 -> MENU.
    • Select your memory channel (e.g., 002).
    • Press MENU -> EXIT.
  6. Save Transmitting Frequency (Critical Step!):
    • Correction for Baofeng: You typically save to the Same Channel again.
    • Press MENU -> 2 7 -> MENU.
    • Select the SAME channel number (e.g., 002).
    • Press MENU to confirm (voice says "Transmitting Memory"), then EXIT.

Essential Menu Settings

Menu Name Setting Description
0 SQL 1-5 Squelch level. Lower is more sensitive.
2 TXP HIGH Transmit power.
7 TDR OFF Dual Watch. Turn OFF for programming to avoid confusion.
25 SFT-D OFF/+/ - Frequency shift direction (for repeaters).
26 OFFSET 00.600 Frequency shift amount (standard 2m offset).
13 T-CTCS OFF/Tone Transmit Continuous Tone Coded Squelch System.

Tips & Troubleshooting

  • Can't enter frequency? Make sure you are in VFO Mode (frequency mode), not Channel mode. Press VFO/MR.
  • Menu not working? You cannot change some settings while in Channel Mode. Switch to Frequency Mode first.
  • "Cancel" when saving? You must Delete the channel (Menu 28) before you can Save to it (Menu 27). Overwriting is not allowed directly.
  • Chirp Software: For programming many channels, we highly recommend using a programming cable and the free CHIRP software.
